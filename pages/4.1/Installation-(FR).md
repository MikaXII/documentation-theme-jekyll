Vous pouvez installer **recalboxOS** sur une carte SD formatée en FAT32.   
**Vous devez utiliser au moins une carte SD de 8GB, et nous recommandons les cartes Sandisk Ultra.**

Utilisez le logiciel de formatage présent ici : https://www.sdcard.org/downloads/formatter_4/ si vous êtes sous Windows ou MacOSX, ou GParted si vous êtes sous Linux.  

* Démarrez le programme **SDCard formatter** et formatez l'intégralité de la carte SD en _**FAT32**_.  
   
_Pour information : si vous utilisez une carte SD de 64Go, il se peut que votre logiciel de formatage force la partition en exFat.   
Si vous rencontrez ce soucis, vous pouvez essayer le logiciel suivant : [minitool partition wizard](http://www.partitionwizard.com/free-partition-manager.html) (seulement compatible windows) choisir partition principale  dans les choix de formatage.

* Téléchargez le fichier recalboxOS.zip dans sa dernière version à l'adresse suivante : https://github.com/digitalLumberjack/recalbox-os/releases  

* Maintenant, décompressez le fichier recalboxOS.zip directement à la racine de votre carte SD.  

* Placez la carte SD dans votre Raspberry Pi, branchez le hdmi et l’alimentation micro-usb.  

* L'installation démarre et **vous aurez une recalbox dans 5 minutes** !   
 
A la fin de l'installation, vous découvrirez l'écran de EmulationStation, et vous aurez le plaisir d'entendre l'excellent thème de Zelda !   

Si vous utilisez un Raspberry Pi 1 modèle B ou B+, vous devrez **vraiment** overclocker celui-ci afin d'obtenir les meilleures performances possibles. Rendez-vous alors dans le menu **Options Système** et modifiez le paramètre d'overclock. 
Je recommande fortement, l'utilisation du réglage d'overclock le plus élevé, **EXTREM**, qui peut certes annuler la garantie constructeur de votre RPi1, mais qui rendra l'expérience de jeu nettement plus agréable que celle obtenue avec les autres réglages.
